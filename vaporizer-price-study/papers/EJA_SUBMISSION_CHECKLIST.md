# EJA Submission Checklist

## Journal Information
- **Journal**: European Journal of Anaesthesiology (EJA)
- **Publisher**: Lippincott Williams & Wilkins / Wolters Kluwer
- **Submission system**: Editorial Manager (https://www.editorialmanager.com/eja/)
- **Cascade journal (if rejected)**: Health Policy

## Blinding
- **NOT double-blind**: Author names and affiliations are included on the title page.
  No need to prepare a blinded version of the manuscript.

## Submission Files

| # | File | Type | Status |
|---|------|------|--------|
| 1 | `eja_manuscript_english.docx` | Manuscript (Title page + Abstract + Text + Acknowledgements + References + Tables + Figure legends) | ✓ Generated |
| 2 | `eja_cover_letter.docx` | Cover Letter | ✓ Generated |
| 3 | `eja_strobe_checklist.docx` | STROBE Checklist (Supplemental) | ✓ Generated |
| 4 | `eja_figures_english.pptx` | Figures (editable, for reference) | ✓ Generated |
| 5 | Figure files (separate PNG/TIFF) | Each figure as separate upload | See `figures/` directory |
| 6 | Copyright Transfer Agreement | Completed electronically after acceptance | N/A at submission |

## Pre-Submission Author Actions Required

### In the manuscript (`eja_manuscript_english.docx`):
- [ ] Fill in author names, affiliations, corresponding author details on Title Page
- [ ] Add ORCID identifiers (if available)
- [ ] Review and finalise acknowledgements
- [ ] Verify all statistics match the data

### Cover letter:
- [ ] Fill in date, corresponding author name, affiliation, and contact details
- [ ] Customise if needed

### Editorial Manager upload:
- [ ] Upload manuscript file
- [ ] Upload each figure as a **separate file** (not embedded in manuscript)
  - Fig. 1: `fig1_price_timeseries.png`
  - Fig. 2: `fig2_boxplot_comparison.png`
  - Fig. 3: `fig3_monthly_median.png`
  - Fig. 4: `fig4_histograms.png`
  - Fig. 5: `fig5_regulatory_timeline.png`
  - Fig. 6: `fig6_quarterly_trends.png`
- [ ] Upload STROBE checklist as Supplemental Digital Content
- [ ] Paste or upload cover letter
- [ ] Enter author information in system
- [ ] Confirm ICMJE authorship criteria for all authors
- [ ] Declare no conflicts of interest

## Format Compliance Summary

| Requirement | EJA Specification | Our manuscript |
|-------------|-------------------|----------------|
| Abstract | Structured, max 300 words | ~260 words ✓ |
| Body text | Max 3500 words | ~2135 words ✓ |
| Line spacing | 1.5 | 1.5 ✓ |
| Margins | ≥ 3 cm | 3 cm ✓ |
| References | Vancouver, superscript after punctuation | 20 refs, superscript ✓ |
| Spelling | UK English | vaporiser, anaesthesia, etc. ✓ |
| Tables | Separate pages | 2 tables + 1 supplementary ✓ |
| Figures | Separate files, max 6 (combined with tables) | 6 figures ✓ |
| Ethics statement | Beginning of Methods | Included ✓ |
| STROBE | Required for observational | Complete checklist ✓ |
| Study design in title | Required | "cross-sectional time-series analysis" ✓ |
| Acknowledgements | "Acknowledgements relating to this article" | Correct heading ✓ |
| Statistics format | P uppercase, n lowercase, no spaces | Compliant ✓ |
| Data format | Mean ± SD, median [IQR] | Compliant ✓ |
| Trial registration | Not applicable (observational) | N/A ✓ |

## Key Notes
- Figures are NOT embedded in the manuscript; only figure legends are included at the end
- Each figure must be uploaded as a separate file on Editorial Manager
- STROBE checklist uploaded as Supplemental Digital Content
- Copyright Transfer Agreement (CTA) is completed electronically (usually after revision or acceptance)
- Visual abstract is optional and can be added at any stage during review
