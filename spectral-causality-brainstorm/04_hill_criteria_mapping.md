# Hill の9基準 × 因果推論手法の包括的マッピング

## 0. Bradford Hill の9基準（1965）

| # | 基準 | 英語 | 内容 |
|---|---|---|---|
| H1 | **強さ** | Strength | 関連の大きさ（効果量） |
| H2 | **一貫性** | Consistency | 異なる集団・状況での再現 |
| H3 | **特異性** | Specificity | 特定の曝露 → 特定のアウトカム |
| H4 | **時間性** | Temporality | 原因が結果に先行 |
| H5 | **量反応関係** | Biological gradient | 用量が増えると反応も増大 |
| H6 | **生物学的妥当性** | Plausibility | メカニズムが生物学的に説明可能 |
| H7 | **整合性** | Coherence | 既知の事実・生物学と矛盾しない |
| H8 | **実験** | Experiment | 介入的エビデンスの有無 |
| H9 | **類似性** | Analogy | 類似の原因が類似の結果を生む |

### 凡例

- **◎**: その基準を直接的・明示的に扱う
- **○**: 部分的に扱う、または適応可能
- **△**: 間接的に関連、または弱い接続
- **—**: 扱わない／該当なし

## 2. 基準別の解説: 各手法が「何をもって」その基準を満たすか

### H1: 強さ（Strength）

効果量・関連の大きさを定量化できるか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 定量的効果量 | RCT, PSM/IPW, IV, MR, DID, RDD, TMLE, DML | ATE/ATT/CATE を推定。信頼区間付き |
| ◎ 因果係数 | LiNGAM系, NOTEARS, GOLEM | 隣接行列 B の係数 b_ij = 直接効果サイズ |
| ◎ 感度分析 | E-value | 「交絡でこの効果が説明されるには、どの程度の未測定交絡が必要か」 |
| ○ 因果指標 | Granger (F統計量), Transfer Entropy (bits) | 統計的有意性は示せるが、因果効果量としての解釈は限定的 |
| **◎ SCI** | スペクトル因果強度 | 磁気ラプラシアンの固有ベクトル荷重+位相差で因果強度を定量化（新規） |
| **○ Utility Causality** | ユーティリティ因果性 | Eigentheme荷重で「テーマレベルの関連強度」。変数レベル効果量ではない |

### H2: 一貫性（Consistency）

異なる集団・設定・手法での再現可能性。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 設計的に担保 | MR（異なるSNPで同じ結果→Consistent）, クロスオーバー（同一被験者で再現） | 手法自体が「異なる条件での再現」を内蔵 |
| ◎ メタ解析的 | GRADE | 複数研究のエビデンスを統合して一貫性を評価 |
| ○ ブートストラップ | LiNGAM bootstrap, Okuda et al.の不確実性定量化 | 同一データ内でのサンプリング安定性 |
| **◎ ECD** | アンサンブル因果方向 | **異なる因果同定原理（非ガウス性, スペクトル, 時間的予測）での一致 = 一貫性** |
| △ ネガティブコントロール | 期待通り帰無を示すことで交絡バイアスの不在を支持 | 直接的に「一貫性」ではないが、結果の信頼性を支持 |

### H3: 特異性（Specificity）

特定の曝露→特定のアウトカムの関係を識別できるか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 変数ペアの因果 | LiNGAM系, PC, FCI, GES, NOTEARS | 隣接行列の (i,j) 成分 = 変数iから変数jへの**特定の**因果エッジ |
| ◎ メカニズム特定 | 因果媒介分析, 四方分解 | 直接効果, 間接効果, 交互作用の分離 → 経路特異性 |
| ◎ 曝露特異的 | RDD（閾値前後の特定曝露のみ） | 極めて局所的・特異的な因果効果 |
| △ テーマレベル | Utility Causality, Eigenthemes | **意図的に特異性を放棄**してテーマレベルで捉える → H3とトレードオフ |
| **○ SCI/SCD** | スペクトル因果 | 変数ペアレベルのSCDも計算可能だが、テーマレベルECIが主目的 |

**重要な洞察**: Utility CausalityはH3（特異性）を**意図的に弱める**ことで、H6（妥当性）、H7（整合性）、H9（類似性）を強める。これは**Hill基準間のトレードオフ**を示唆している。

### H4: 時間性（Temporality）

原因が結果に時間的に先行することを保証/活用するか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 設計で保証 | RCT（介入が先）, DID（政策変更が先）, ITS（介入時点が明確） | 研究デザイン自体が時間的先行を保証 |
| ◎ 時間構造を利用 | VARLiNGAM, Granger, Transfer Entropy, PCMCI+ | 時間遅れ構造を明示的にモデル化 |
| ◎ 遺伝的に保証 | MR | 遺伝型は出生時に決定 → 曝露・アウトカムに先行 |
| ◎ ワークフロー制約 | Longitudinal LiNGAM (Okuda+) | 測定プロトコルの時間順序 → 許容エッジ制約 |
| ◎ Hodge勾配 | Hodge-Causal Decomposition | **ポテンシャル φ(i) の勾配方向 = 時間的・因果的フロー方向** |
| ○ 磁気位相 | SCI/SCD | 磁気ラプラシアンの位相差に時間方向を符号化 |
| — 横断のみ | DirectLiNGAM, PC, PSM | 横断データでは時間的先行を**保証できない** |

### H5: 量反応関係（Biological Gradient）

用量-反応関係（曝露量↑→効果量↑）をモデル化できるか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 柔軟な非線形 | CAM-UV (GAM), TMLE, DML, G-computation | 非線形の用量-反応曲線を推定可能 |
| ◎ 閾値効果 | RDD | 閾値前後での不連続な効果 = 特殊な量反応関係 |
| ◎ 連続処置 | 連続処置PSM, GPS (Generalized Propensity Score) | 連続値の「用量」に対する反応曲線 |
| △ 線形のみ | LiNGAM系（基本版） | b_ij は線形係数 → 用量反応は線形と仮定 |
| **○ Utility Causality** | ユーティリティ因果性 | LLMが「HbA1c 7.0 vs 10.0」の臨床的意味の違いをユーティリティ質問に反映 → **暗黙的に量反応を捕捉** |
| — 構造のみ | PC, FCI, GES | エッジの有無のみで効果量なし |

### H6: 生物学的妥当性（Plausibility）

メカニズムが生物学的に説明可能であるという知識を組み込めるか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| **◎ 知識組込み** | DAG（専門家構築）, 十分原因モデル, 因果媒介分析 | メカニズムの仮説を**グラフ構造**として明示的に表現 |
| **◎ LLM知識** | **Utility Causality** | **臨床知識をLLMのユーティリティ質問生成を通じてグラフ構造に注入** |
| ○ 事前知識 | DirectLiNGAM (prior_knowledge引数), Bayesian DAG | 事前分布・制約としてドメイン知識を投入 |
| ○ 生物学的根拠 | MR | 遺伝子 → 曝露 → アウトカムの生物学的パスウェイ |
| ○ ワークフロー知識 | Longitudinal LiNGAM (Okuda+) | 臨床プロトコルの測定順序 = ある種の生物学的制約 |
| — データ駆動のみ | PC, Granger, NOTEARS, Transfer Entropy | ドメイン知識を組み込む仕組みがない |

**本スレッドの最大の発見の一つ**: Utility Causalityは**H6（妥当性）に最も強い手法**。LLMを介して臨床知識をグラフに構造的に注入できるのは、他の統計的手法にはない特徴。

### H7: 整合性（Coherence）

既知の自然史・生物学と矛盾しないか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| **◎ 知識との照合** | DAG（既知生物学で検証）, 十分原因モデル, GRADE | 結果が既知の知識体系と整合するか明示的に評価 |
| **◎ Utility整合性** | **Utility Causality / Eigenthemes** | Eigenthemeが既知の疾患概念と一致するか → **自動的に整合性チェック** |
| **◎ 手法間整合性** | **ECD** | 異なる手法の結果が一致する = 「整合的」。不一致 = 調査必要 |
| ○ ネガティブコントロール | Negative control exposure/outcome | 既知の非因果関係でバイアスの有無を検証 → 整合性の間接的確認 |
| ○ ワークフロー整合 | Longitudinal LiNGAM (Okuda+) | 結果が測定プロトコルと整合するか確認 |
| ○ スペクトル連結性 | Spectral regularized DAG (M'Charrak+) | 「科学的グラフは連結であるべき」= 科学的整合性の帰納バイアス |

### H8: 実験（Experiment）

介入的（or 準介入的）エビデンスを提供するか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| ◎ 真の介入 | RCT, クロスオーバー, N-of-1 | ランダム化された介入 = 最強の実験的証拠 |
| ◎ 介入的解釈 | do-calculus / SCM | do(X) 演算子 = 介入の数学的定義 |
| ◎ 反事実 | 潜在結果モデル (Rubin) | Y(1) - Y(0) = 個体レベル因果効果（観察不可能だが定義） |
| ○ 準実験 | IV, MR, DID, RDD, ITS, Synthetic Control | 自然実験・制度変更・閾値を「疑似介入」として活用 |
| ○ SEM介入解釈 | LiNGAM系 | SEMの構造 → do演算子との対応 → 介入効果の推定が理論上可能 |
| — 情報的のみ | Granger, Transfer Entropy, CCM | 予測的関連であり介入的解釈はない |
| **— 情報的因果** | **Utility Causality, SCI/SCD, Hodge** | **「何が何に役立つか」= 情報的関連。介入的因果ではない** |

**核心的位置づけ**: 提案手法群はH8（実験）を**意図的にカバーしない**。これはPearlの因果の梯子（Ladder of Causation）で言えば、第1段（Association）と第2段（Intervention）の**間**に位置する新しい概念。

### H9: 類似性（Analogy）

類似の原因→類似の結果というパターンを活用するか。

| 評価 | 手法群 | どう扱うか |
|---|---|---|
| **◎ パターン類似性** | **GEM-RAG Eigenthemes** | **「同じユーティリティ質問に答える」= 類似パターンの体系的検出** |
| **◎ 臨床テーマ類似** | **Utility Causality** | **同じEigenthemeに荷重 = 「類似の因果メカニズム」の数学的表現** |
| ◎ 制御群類似性 | 合成コントロール法 | 処置群に「類似」した合成対照を構築 |
| ◎ 成分類似性 | 十分原因モデル (Rothman) | 「成分原因のパターンが類似」= 類似の因果メカニズム |
| ○ MR多重道具 | MR | 異なるSNPが同じ結果 → 「異なる経路だが類似の因果」 |
| ○ アトラクター類似 | CCM | 力学系のアトラクターの類似構造 |
| ○ スペクトル類似 | 因果フーリエ解析 (Seifert+) | スペクトルドメインでの類似構造 |
| ○ 傾向類似 | PSM | 傾向スコアの近さ = 「交絡構造が類似」 |
| — | LiNGAM, Granger, DID, RDD等 | 類似性の概念を直接利用しない |

**本スレッドの最大の発見の二つ目**: Utility CausalityとEigenthemesはH9（類似性）を**最も直接的に扱う手法**。Hill自身がH9を最も「弱い」基準とし他が具体的な方法論を欠いていたのに対し、スペクトル手法はH9を**計算可能にする**。

## 4. 核心的洞察

### 4-1. 既存手法は H1, H3, H4, H8 に集中している

ほとんどの統計的因果推論手法は、Hill基準のうち：
- **H1（強さ）**: 効果量の定量化
- **H3（特異性）**: 変数ペアの特定
- **H4（時間性）**: 時間的先行
- **H8（実験）**: 介入的解釈

を扱うよう設計されている。これは「反事実・介入主義パラダイム」の反映。

### 4-2. H6, H7, H9 は「手薄」だった

- **H6（妥当性）**: DAGの手描きか、事前知識引数でしか入れられなかった
- **H7（整合性）**: 主に研究者の判断に委ねられ、計算的手法がなかった
- **H9（類似性）**: Hill自身が最も弱い基準としたが、**計算的に扱う手法がほぼ存在しなかった**

### 4-3. 提案手法群は H6, H7, H9 を埋める

```
既存手法の得意領域:  H1 ◎  H3 ◎  H4 ◎  H8 ◎
                     ↕ 補完的 ↕
提案手法の得意領域:  H6 ◎  H7 ◎  H9 ◎
```

**Utility Causality** は LLM を介して臨床知識をグラフ構造に注入し → **H6（妥当性）を計算化**
**ECD** は異なる手法の一致を体系的にチェックし → **H7（整合性）を計算化**
**Eigenthemes** は類似パターンをスペクトル分解で検出し → **H9（類似性）を計算化**

### 4-4. Hill基準間のトレードオフの発見

```
特異性 (H3) ←→ 類似性 (H9): トレードオフ
  変数レベル因果（LiNGAM）は H3◎ だが H9—
  テーマレベル因果（Utility Causality）は H3△ だが H9◎

実験 (H8) ←→ 妥当性 (H6): 独立
  RCTは H8◎ だが H6△（メカニズムは問わない）
  Utility Causalityは H8— だが H6◎（メカニズムの知識を活用）

→ 「全基準を一つの手法で満たす」のは不可能
→ ECD的なアンサンブルで複数手法を組み合わせることが必然
```

### 4-5. Pearl の因果の梯子との対応

```
Level 3: 反事実 (Counterfactual)  → Rubin, do-calculus  [H8 ◎]
Level 2: 介入 (Intervention)     → RCT, IV, MR, DID    [H8 ○, H1 ◎]
Level 1: 関連 (Association)      → Granger, 相関       [H1 ○]

新提案:
Level 1.5: 情報的因果 (Informational Causality)
  = 「何が何の理解に役立つか」
  → Utility Causality, SCI/SCD
  [H6 ◎, H7 ◎, H9 ◎]

Pearlの梯子にはないこの中間層が、
まさにHillが直感的に H6/H7/H9 で捉えようとしたもの
```

## 5. まとめ: Hill基準は「因果推論の方法選択ガイド」にもなる

Hill は9基準を「因果判断のチェックリスト」として提案したが、裏返すと：

> **「どの基準のエビデンスが不足しているか」を特定し、その基準を扱える手法を選択するガイド**

として使える。

```
欠けているエビデンス → 選択すべき手法

H1（効果量が不明）    → RCT, LiNGAM, PSM, SCI
H2（再現性が不明）    → MR（多重IV）, ECD, メタ分析
H3（特異性が不明）    → LiNGAM, 因果媒介分析, PC/FCI
H4（時間順序が不明）  → VARLiNGAM, Granger, DID, Hodge分解
H5（量反応が不明）    → CAM-UV, TMLE, DML, RDD
H6（妥当性が不明）    → DAG構築, Utility Causality, 因果媒介
H7（整合性が不明）    → ECD, ネガティブコントロール, Eigenthemes
H8（実験的証拠なし）  → MR, IV, 自然実験デザイン
H9（類似症例が不明）  → Utility Causality, Eigenthemes, 合成コントロール
```
