# CNIPシミュレーション報告書: 周術期アウトカム試験

## 1. シミュレーション設計

### シナリオ
大腸手術後の術後合併症に対する Enhanced Recovery Protocol (ERP) vs 標準治療のランダム化比較試験（RCT）（n=2000、1:1割付）。

### アウトカム
合併症重症度指数（連続値、0-1スケール）:

```
Y_obs(i) = 0.50 + S(i) + N_conf(i) + N_bio(i) + N_meas(i) + N_rand(i)
```

### 真のパラメータ（Ground Truth）

| 成分 | 説明 | パラメータ |
|------|------|------------|
| **S（シグナル）** | 治療効果 | 基本ATE = -0.15; 高齢者（>75歳）追加効果 = -0.10 |
| **N_conf** | 測定可能な交絡因子 | 年齢(0.012/歳), BMI(0.006/単位), ASA-PS(0.10/段階), 糖尿病(+0.12), 手術時間(0.0008/分) |
| **N_bio** | 生物学的変動 | CRP(0.003/mg/L), ヘモグロビン(-0.04/g/dL) |
| **N_meas** | 測定誤差 | 施設間ランダム効果（SD=0.06、10施設） |
| **N_rand** | 還元不能ノイズ | Normal(0, 0.25) |

### 生物学的根拠
- **年齢**: 最も強い交絡因子 — 高齢患者は生理的予備能低下、創傷治癒遅延、合併症感受性上昇
- **ASA-PS**: 全体的な合併症負荷を反映; ASA 3-4は有意に周術期リスク高い
- **ヘモグロビン**: 術前体力と酸素運搬能を反映; 貧血は確立されたリスク因子
- **CRP**: ベースライン炎症マーカー; 高CRPは慢性炎症状態を示唆
- **手術時間**: 手術の複雑さと術中ストレスの代替指標
- **高齢者の追加効果**: ERPプロトコルには早期離床・栄養最適化が含まれ、不動関連合併症に脆弱な高齢者がより大きな恩恵を受ける — 生物学的に妥当

## 2. CNIPパイプライン実装

### Phase 1: ノイズ前方モデル
- **モデル**: Gradient Boosting Regressor（200本、max_depth=3、min_samples_leaf=20）
- **学習**: コントロール群のみ（n=987）— 治療効果が存在しないためシグナル漏洩を防止
- **予測**: コントロール群は交差検証（5-fold）、治療群はコントロール群全体で学習したモデルで予測
- **上位特徴量**: 年齢(40.1%), ヘモグロビン(15.7%), 手術時間(13.9%), CRP(6.2%), 糖尿病(5.8%)

### Phase 2: ノイズパラメータ推定
- **指標**: コントロール群における交差検証R二乗（過学習なしの正直な推定）
- **不確実性**: 1000回ブートストラップリサンプリングで95%信頼区間
- **結果**: rho二乗 = 0.324（95% CI: [0.263, 0.376]）

### Phase 3: 残差生成
- 残差 = Y_obs - F(theta_hat; X)
- 分散削減: 29.9%（SD: 0.327 → 0.274）

### Phase 4: 仮説フリー発見
- 残差からの治療効果推定
- サブグループ解析（年齢層別）
- 治療×年齢の交互作用検定

## 3. 結果

### 治療効果回復

| 手法 | ATE推定値 | 標準誤差 | 95%信頼区間 | p値 |
|------|-----------|----------|-------------|-----|
| 真の値 | -0.150 | — | — | — |
| ナイーブ | -0.168 | 0.0141 | [-0.196, -0.140] | <0.001 |
| CNIP | -0.158 | 0.0117 | [-0.181, -0.135] | <0.001 |

- CNIPバイアス: 0.008（vs ナイーブバイアス: 0.018）— **56%のバイアス削減**
- 標準誤差削減: 17.1%
- 両手法とも有意差を検出するが、CNIPはより狭い信頼区間を提供

### サブグループ発見

| サブグループ | n | CNIP ATE | 標準誤差 | 真のATE |
|-------------|---|----------|----------|---------|
| 高齢者（>75歳） | 411 | -0.238 | 0.0255 | -0.250 |
| 非高齢者（≦75歳） | 1589 | -0.137 | 0.0131 | -0.150 |

- 交互作用係数: -0.100（真の値: -0.100）— **優れた回復精度**
- CNIPは高齢患者がERPからより大きな利益を得ることを正確に同定

### 検出力比較

| n | ナイーブ検出力 | CNIP検出力 | 改善幅 |
|---|---------------|-----------|--------|
| 100 | 72.2% | 87.8% | +15.6ポイント |
| 200 | 98.0% | 99.6% | +1.6ポイント |
| 400 | 100% | 100% | — |

- n=100でCNIPは87.8%の検出力を達成（ナイーブ72.2%）— 小規模試験で決定的な差
- CNIPは約n=80で80%検出力に到達、ナイーブは約n=120 — **約33%のサンプルサイズ削減**

## 5. 生物学的整合性の評価

### 特徴量重要度が臨床知見と一致
ノイズモデルの特徴量ランキング（年齢 > ヘモグロビン > 手術時間 > CRP > 糖尿病）は確立された周術期リスク文献と一致:
- **年齢**が最も支配的な予測因子であることは、フレイルティと生理的予備能の概念と合致
- **ヘモグロビン**は術前貧血の手術転帰への既知の影響を反映
- **手術時間**は複雑さの代替指標として標準的リスク因子
- **CRPと糖尿病**は既知の炎症性・代謝性リスク経路と一致

### サブグループ効果の生物学的妥当性
高齢患者がERPからより大きな利益を得るという知見は以下と整合:
- 早期離床は、不動関連合併症（DVT、肺炎）に対してより脆弱な高齢患者でより効果的
- 栄養最適化は、高齢外科患者でより高頻度の術前低栄養を改善
- マルチモーダル鎮痛はオピオイド関連合併症（せん妄、呼吸抑制）を削減 — 高齢者で特に重要

### ノイズモデル精度（rho二乗 = 0.32）の現実性
アウトカム分散の約32%を説明するノイズモデルは、周術期医学における予後モデルの文献値（典型的にR二乗 = 0.2-0.4）と一致し、本シミュレーションが生物学的に現実的なノイズ構造を使用していることを確認。

## 6. 結論

1. **CNIPフレームワークは真の治療効果を正確に回復** — ナイーブ解析より低バイアス・狭信頼区間
2. **ノイズモデルは現実的な精度を達成** — rho二乗=0.32は臨床予後モデルと同水準
3. **残差の脱相関が機能** — 交絡因子との相関は実質的に除去（最大|r|=0.067）
4. **サンプルサイズ効率は意味のある大きさ** — 1.48倍の有効サンプル、約33%少ない患者数
5. **サブグループ発見が成功** — 植え込まれた高齢者効果の異質性を正確に同定
6. **全結果が生物学的に整合** — 確立された周術期医学文献と一致

### 限界
- シミュレーションは連続アウトカムを使用; 二値アウトカムでは性能が異なる可能性
- 検出力シミュレーションでは線形回帰を使用（本パイプラインの勾配ブースティングと異なる）
- 単一の乱数シード（42）; 複数シードでの確認が必要
- 実臨床データには欠損値、時変交絡因子、より複雑なノイズ構造が存在
