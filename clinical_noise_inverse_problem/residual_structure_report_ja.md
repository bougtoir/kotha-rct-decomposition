# 残差構造分析: CNIPの独自価値の検証

## 問い

ANCOVAなどの既存手法はATEの点推定で終わる。CNIPの残差構造分析は、それ以上の情報を引き出せるか？

---

## 5つのシナリオと結果

### シナリオ1: サブグループ効果の不均一性（HTE）

**設定**: n=2000のRCT。真のATEは年齢・バイオマーカーで異なる:
- 高齢者（Age>75）: ATE = -0.30（強い治療効果）
- 高バイオマーカー（>1）: ATE = -0.20（中程度の効果）
- その他: ATE = 0.00（効果なし）
- 全体平均ATE ≈ -0.09

**結果**:

| 手法 | 出力 |
|------|------|
| ANCOVA | ATE = **-0.082**（単一の平均値） |
| 残差構造分析（年齢層別） | Age<55: -0.021 (ns) / Age 55-65: -0.010 (ns) / Age 65-75: -0.022 (ns) / **Age>75: -0.320 (p<0.001)** |
| 残差構造分析（バイオマーカー層別） | Bio<-1: -0.043 (ns) / Bio -1~0: -0.079 (p<0.001) / Bio 0~1: -0.080 (p<0.001) / **Bio>1: -0.144 (p<0.001)** |

→ **ANCOVAのATE=-0.082は「全員に控えめな効果」と読めるが、実際には高齢者に-0.320の強い効果、若年者にはほぼ効果なし。** この不均一性は残差分析なしでは見えない。

### シナリオ2: 非線形用量反応曲線

**設定**: n=2000。用量依存的な治療効果（対数曲線: 低用量で急峻、高用量で飽和）。

**結果**:

| 用量 | 残差から推定した効果 | 真の効果 |
|------|-------------------|---------|
| 0.15 | -0.077 | -0.092 |
| 0.45 | -0.192 | -0.213 |
| 0.75 | -0.347 | -0.295 |
| 0.95 | -0.333 | -0.337 |
| **ANCOVA** | **-0.227（全用量平均）** | — |

→ **ANCOVAは-0.227という単一値のみ。残差分析は用量反応曲線を再構成し、「低用量でも一定の効果がある」「高用量では飽和する」という臨床的に重要な情報を抽出。**

### シナリオ3: 非測定交絡因子の検出

**設定**: n=2000のRCT。遺伝的バリアント（測定されていない、有病率30%）がアウトカムに影響。

**結果**:

| 指標 | ANCOVA残差 | CNIP残差 |
|------|-----------|---------|
| 二峰性スコア | 2.66 | 2.68 |
| クラスター vs 真のバリアント ARI | — | 0.116 |

→ **正直な評価: 残差の二峰性は検出されたが、クラスタリングと真の遺伝的バリアントの一致度は低い（ARI=0.116）。** 残差から非測定交絡の「存在」を示唆できるが、「特定」は困難。これは残差構造分析の限界であり、論文では限界として明記すべき。

### シナリオ4: 治療効果の個別化マップ（HTE表面）

**設定**: n=2000。ATEが年齢×BMIの連続関数として変動。

**結果**:
- ANCOVA: ATE = -0.050（全患者で単一値）
- 残差から推定したHTEマップと真のHTEマップの相関: **r = 0.557**

→ **残差分析は共変量空間上のATE表面をある程度再構成できる。** 完全ではないが（r=0.557）、「どの患者が最も効果を受けるか」の方向性を示す。精密医療への橋渡しとなる情報。

### シナリオ5: サブグループ検出率のモンテカルロ比較

**設定**: 500回MC。n=500。高齢者（Age>70）にATE=-0.25、若年者にATE=0。

**結果**:
- ANCOVAの交互作用検定: 検出率 **100%**
- CNIP残差分析: 検出率 **100%**

→ **この設定では効果量が大きすぎて両手法とも100%検出。** より小さい効果量（例: ATE=-0.10）で再検証が必要。ただし重要な点: **ANCOVAの交互作用検定は「どの変数で分割するか」を事前に指定する必要がある。** 残差分析は事前指定なしで網羅的にスキャンできる。

## 正直な総括

### 残差構造分析が明確に有用なシナリオ

| シナリオ | 有用性 | 判定 |
|---------|--------|------|
| **サブグループHTE** | ANCOVAが見逃す治療効果の不均一性を検出 | **有用** |
| **用量反応曲線** | 二値ANCOVAでは得られない曲線を再構成 | **有用** |
| HTE表面マップ | 方向性は捉えるが精度は中程度（r=0.557） | **部分的に有用** |
| 非測定交絡検出 | 存在の示唆のみ、特定は困難（ARI=0.116） | **限定的** |
| 検出率比較 | 大効果量では差なし。事前指定不要が利点 | **条件付き** |

### 重要な注意点

1. **ANCOVAの交互作用検定でもHTEは検出できる** — ただし検定する交互作用を事前に指定する必要がある（多重比較の問題）。残差分析の利点は「事前仮説なしの探索」。

2. **残差分析はexploratory（探索的）であり、confirmatory（確証的）ではない** — 発見された効果は独立データセットでの検証が必要。論文では「hypothesis-generating（仮説生成）」として位置付けるべき。

3. **Phase 4を「仮説フリー発見」と呼ばない方がよい** — 医学界のコンサバ性を考慮し、「残差構造の系統的分析（systematic residual structure analysis）」と呼ぶ方が受容されやすい。

## 論文への組み込み方

### 推奨: 「残差構造分析」を論文の第2の柱にする

**柱1**: 既存手法の統一フレームワーク（Y = S + N）
**柱2**: 残差構造分析（既存手法に欠けている機能）

### 具体的なセクション構成

```
4. Results
  4.1 統一フレームワークによる既存手法の比較
      → ANCOVAが効率性で優位であることを正直に示す
  4.2 残差構造分析: ATE推定を超える情報
      4.2.1 サブグループ効果の不均一性検出
      4.2.2 用量反応曲線の再構成
      4.2.3 治療効果の個別化マップ
  4.3 限界: 非測定交絡の検出は困難
```

### フレーミング

> 「CNIPフレームワークは、既存の共変量調整手法を統一的に記述するだけでなく、
> 残差の構造を系統的に分析する手法を提供する。これにより、ATEの点推定では
> 失われる治療効果の不均一性（HTE）、用量反応関係、および個別化治療効果
> マップを探索的に検出できる。この残差構造分析は、既存のどの共変量調整手法
> にも組み込まれていない機能であり、仮説生成ツールとして既存手法を補完する。」

### 用語の選択

| 避けるべき表現 | 推奨する表現 | 理由 |
|--------------|------------|------|
| 仮説フリー発見 | 系統的残差構造分析 | コンサバな医学界での受容性 |
| CNIPはANCOVAに勝る | CNIPはANCOVAを補完する | シミュレーションで劣る点がある |
| 新しい統計手法 | 統一フレームワーク + 残差分析 | 既存手法の再構成であることを明確に |
| Phase 4 | 残差の探索的分析 | 専門用語を避ける |
